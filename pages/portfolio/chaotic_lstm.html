This post offers a quick overview of my dissertation work on computer-assisted proofs of chaos in machine learning models. If you are here, I assume you have some background in math and machine learning.

1. Preliminaries

This work combines three distinct areas of mathematics: machine learning, dynamical systems and the study of chaos, and computer assisted proofs via rigorous computation.

1.1. LSTM

The Long Short-Term Memory model (LSTM) was developed to mitigate the vanishing gradient problem. This occurs when using recurrent neural networks (RNN), which infer using a long composition of the same function. Via the chain rule, this long composition can have derivatives tend to zero in absolute value resulting in underflow in computation. The result is that an RNN will fail to learn long range patterns. To see this, consider \(f(x_t, h_{t-1}) = h_t\) where \(x\) is an input sequence and \(h\) is a hidden state. Then \[\dfrac{\partial h_n}{\partial x_1} = \dfrac{\partial h_1}{x_1} \prod_{i = 1}^{n-1} \dfrac{\partial h_{i+1}}{\partial h_{i}}\] Thus, this large product and vanish (or explode) and the dependencies of the \(n^{\text{th}}\) hidden state won't depend on the initial input.

1.2. Chaos

For a family of dynamical system \(f_{\theta}: X \to X\), with parameter \(\theta\) we are interested in the limit behavior of the system. We may be interested in the set \(\Omega(x_0) = \limsup_{n \to \infty} \{x \in X : x = f_{\theta}^n(x_0), n \in \mathbb{N}\}\), the omega limit set. This could be a single point or something more interesting. We could also be interested in what happens to these sets as \(\theta\) changes.
