1. Motivation

We first return to the core problem in machine learning. Given data \(X\), a family of models \(f_{\theta}\) and a loss function \(L(X,\theta)\) we wish to solve

\begin{align*} f_{\theta_*} &= \text{argmin}_{f_{\theta}} L(X, f_{\theta}). \end{align*}

And the most popular way to solve this optimization problem is via gradient descent. But we can also find minima using Newton-Raphson. The update rule for Newton-Raphsonn is

\begin{align*} x_{n+1} &= x_n - \alpha \frac{g(x_n)}{g'(x_n)} \end{align*}

where \(g\) is a function we are trying to find a zero of. This Newton-Raphson update is a step towards a root. In context, that is

\begin{align*} f_{n+1} &= f_n - \alpha \frac{L'(X,f_n)}{L''(X,f_n)} \end{align*}

Instead of using this update directly, we'll approximate it with another model

\begin{align*} f_{n+1} &= f_n - \alpha \hat{\phi}_{n+1} \end{align*}

where \(\hat{\phi}_{n+1}\) learns the difference between \(f_n\) and the true data/outputs. In the end, \(f_{N}\) will be a sum of models. In particular,

\begin{align*} \hat{\phi}_{n+1} &= \text{argmin} \sum_{i = 1}^N \frac{1}{2} \hat{h}_{n+1}(x_i) \left [\phi(x_i) - \frac{\hat{g}_{n+1}(x_i)}{\hat{h}_{n+1}(x_i)} \right ]^2\\ \hat{g}_{n+1}(x_i) &= \frac{\partial L(y_i, f_n(x_i))}{\partial f_n(x_i)}\\ \hat{h}_{n+1}(x_i) &= \frac{\partial^2 L(y_i, f_n(x_i))}{\partial f_n(x_i)^2} \end{align*}
