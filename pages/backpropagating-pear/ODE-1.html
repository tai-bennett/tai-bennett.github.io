In this section, we consider the problem

\begin{align*} \frac{d \vec{y}}{dt} &= \vec{f}(t, \vec{y}), && \vec{y}(t_0) = \vec{c}_0 && \text{(P)} \end{align*}

which we label (P). We will present some fundamental theorems regarding solutions to (P) in cases such that \(f\) is Lipschitz on a rectangle, \(f\) is just continuous. Lastly, we discuss some results regarding the expansion of solutions.

1. Existence and uniqueness with the Lipschitz condition

Let \(| \cdot | = \max \{|x_i| : 1 \leq i \leq n\}\) be the sup-norm on \(\mathbb{R}^n\). We will prove the existence and uniqueness of solutions to (P) when \(f\) is Lipschitz continuous on a rectangle. We work with two main assumptions:

  • Assumption 1 Each component function \(f^{(i)}(t, \vec{y})\) of \(\vec{f}\) is a real-valued, continuous function on a rectangular region: \[\mathcal{R} = \mathcal{R}((t, \vec{c}_0), a, b) = \{(t, \vec{y}) : |t - t_0| \leq a, \ |\vec{y} - \vec{c}_0| \leq b\}.\] (note the use of the sup-norm) where \(a,b\) are two positive numbers.
  • Assumption 2 The function \(\vec{f}\) is Lipschitz with constant \(L >0\). That is, \[| \vec{f}(t, \vec{y}_1) - \vec{f}(t, \vec{y}_2) | \leq L |\vec{y}_1 - \vec{y}_2|\] where \((t,\vec{y}_j) \in \mathcal{R}\) for \(j = 1,2\).

Then, we have the following theorem.

Theorem (FTODE version 1) Unders assumptions 1 and 2, there exists a unique solution of problem (P) on the interval \(|t - t_0| < \alpha\) where \(\alpha = \min \{a,b/M\}\) and \(M = \max_{(t,\vec{y}) \in \mathcal{R}} | \vec{f}(t, \vec{y}) |\).

This means that the problem (P) has a local solution (in time) that is unique whenever \(f\) is smooth enough (Lipschitz). There are two keys to the proof: Gronwall's inequality and a method of successive approximations to the solution.

1.1. Gronwall's Inequality

Theorem (Gronwall's Inequality) Let \(g(t)\) be a continuous function on \([t_0, t_1]\) such that

\begin{align} 0 \leq g(t) \leq K + L \int_{t_0}^t g(s) ds \label{eqn:gronwall-cond} \end{align}

for all \(t \in [t_0, t_1]\) then

\begin{align} 0 \leq g(t) \leq K \exp (L(t-t_0)) \label{eqn:gronwall-ineq} \end{align}

for \(t \in [t_0, t_1]\).

Proof.

Let \(v(t) = \int_{t_0}^t g(s) ds\) and substituting into (\ref{eqn:gronwall-cond}) we get that \(v'(t) \leq K + Lv(t)\) and \(v(t_0) = 0\). Then we can write

\begin{align*} -Lv(t) + v'(t) & \leq K\\ \exp(-L(t-t_0))(-Lv(t) + v'(t)) & \leq K \exp(-L(t-t_0))\\ -L \exp (-L (t-t_0)) v(t) + \exp(-L(t-t_0)) v'(t) & \leq K \exp(-L(t-t_0))\\ \frac{d}{dt} \left ( \exp ( -L(t-t_0) ) v(t) \right ) &\leq K \exp (-L(t-t_0))\\ \int_{t_0}^t \frac{d}{dt} \left ( \exp ( -L(s-t_0) ) v(s) \right ) ds &\leq \int_{t_0}^t K \exp (-L(s-t_0)) ds\\ \exp ( -L(t-t_0) ) v(t) &\leq \frac{K}{L} \left ( 1 - \exp (-L(t-t_0)) \right ) \end{align*}

Solving the above for \(Lv(t)\), we can conclude that \[ g(t) \leq K + Lv(t) \leq K \exp (L(t-t_0)). \]

\(\square\)

1.2. Successive Approximations of a solution

Since we are restricting our focus of \(\vec{f}\) on a compact domain, we can assume absolute continuity, meaning that for solutions we must have the fundamental theorem of calculus. That is

\begin{align*} \vec{\phi}(t) - \vec{c}_0 &= \int_{t_0}^t \vec{\phi}'(s) ds\\ &= \int_{t_0}^t \vec{f}(s, \phi(s)) ds. \end{align*}

Meaning that solutions must be a fixed point of the mapping \(\vec{\psi}(t) \mapsto \vec{c}_0 + \int_{t_0}^t \vec{f}(t, \vec{\psi}(s)) ds\). This inspires our approximation method of our solution. We start with a simple function \(\psi\) and iterate this mapping until we reach a fixed point.

In particular, we consider the sequence

\begin{align*} \begin{cases} \vec{\phi}_0(t) = \vec{c}_0 & \\ \vec{\phi}_{k+1}(t) = \vec{c}_0 + \int_{t_0}^t \vec{f}(s, \vec{\phi}_{k}(s)) ds, & k = 0, 1, 2 \ldots. \end{cases} \end{align*}

1.3. Proof of FTODE version 1

The core of the proof of this version of the fundamental theorem is that our sequence of approximations \(\{\vec{\phi}_k(t)\}\) converges (uniformly on \([t_0 - \alpha, t_0 + \alpha]\)) and then confirming that this limit function is a unique solution to (P). The uniqueness can be proved by applying Gronwall's inequality of the difference of two solutions.

To show the convergence of \(\vec{\phi}_k(t) = \vec{\phi}_0(t) + \sum_{i = 1}^i (\vec{\phi}_i(t) - \vec{\phi}_{i-1}(t))\) can be shown in several small steps. First, by induction, one can show

\begin{align*} |\vec{\phi}_k(t) - \vec{\phi}_{k-1}(t)| \leq \frac{M L^{k-1}}{k!} |t - t_0|^{k+1} \end{align*}

where \(L\) is the Lipschitz constant of \(\vec{f}\). This can be used to show the uniform convergences of the series \(\sum_{k = 1}^{\infty} (\vec{\phi}_k(t) - \vec{\phi}_{k-1}(t))\) on \([t_0 - \alpha, t_0 + \alpha]\) which in turn shows the uniform convergence of \(\vec{\phi}_k(t)\) on the same domain. Denote the limit function by \(\vec{\phi} : [t_0 - \alpha, t_0 + \alpha] \to D\).

The uniform convergence means we can pass the limit

\begin{align*} \vec{\phi}_k(t) &= \vec{c}_0 + \int_{t_0}^t \vec{f}(s, \vec{\phi}_{k-1}(t)) ds \\ \lim_{k \to \infty} \vec{\phi}_k(t) &= \vec{c}_0 + \lim_{k \to \infty} \int_{t_0}^t \vec{f}(s, \vec{\phi}_{k-1}(t)) ds \\ \lim_{k \to \infty} \vec{\phi}_k(t) &= \vec{c}_0 + \int_{t_0}^t \lim_{k \to \infty} \vec{f}(s, \vec{\phi}_{k-1}(t)) ds \\ \vec{\phi}(t) &= \vec{c}_0 + \int_{t_0}^t \vec{f}(s, \vec{\phi}(t)) ds \end{align*}

which is clearly a solution to (P).

To show uniqueness, suppose \(\vec{\psi}(t)\) is another solution on \([t_0 - \alpha', t_0 + \alpha']\) for \(\alpha' \leq \alpha\). Then applying Gronwall's inequality (with \(K = 0\)) to

\begin{align*} | \vec{\phi}(t) - \vec{\psi}(t) | = \left | \int_{t_0}^t \left ( \vec{f}(s, \vec{\phi}(s)) - \vec{f}(s, \vec{\psi}(s)) \right ) ds \right | \leq L \left | \int_{t_0}^t | \vec{\phi}(s) - \vec{\psi}(s) | ds \right | \end{align*}

we get

\begin{align*} | \vec{\phi}(t) - \vec{\psi}(t) | \leq K \exp (L (t-t_0)) = 0 \end{align*}

which confirms uniqueness.

\(\square\)

2. Existence without the Lipschitz condition

This proof depends on our ability to approximate continuous functions with smooth function which is a result of Arzela-Ascoli. The idea of the proof is to approximate \(\vec{f}\) with smooth functions \(\vec{F}_{\epsilon}\) (smooth implies Lipschitz) and thus have solutions \(\vec{\phi}_{\epsilon}(t)\) to the initial value problem. These approximations take place under the sup-norm. The solutions \(\{\vec{\phi}(t) : \epsilon > 0\}\) form a bounded, equicontinuous set which an be used to get our solution via Arzela-Ascoli.

2.1. Arzela-Ascoli

Theorem (Arzela-Ascoli) On a bounded interval \(\mathcal{I}\), let \(\mathcal{F}\) be an infinite, bounded and equicontinous set of functions. Then \(\mathcal{F}\) contains an infinite sequence which is uniformly convergent on \(\mathcal{I}\).

Proof.

Let \(\mathbb{Q} \cap \mathcal{I} =: \mathcal{Q} = \{r_j : j \in \mathbb{N}\}\). That is, the set \(\mathcal{Q}\) is the set of rational numbers in our interval and we enumerate them.

We start by constructing a sequence that converges pointwise on \(\mathcal{Q}\). We can do this with a diagonalization argument. Let \(\mathcal{F}_1\) be a subsequence of \(\mathcal{F}\) such that \(f_{1,\ell}(r_1)\) converges to some \(c_1\) as \(\ell \to \infty\). Then inductively define \(\mathcal{F}_j\) as a subsequence of \(\mathcal{F}_{j-1}\) that converges at \(r_j\) to \(c_j\) (that is \(f_{j, \ell}(r_j) \to c_j\) as \(\ell \to \infty\)). Then the sequence \(f_{h,h}\) converges for all rational point in the interval. We can now upgrade this convergence on \(\mathcal{Q}\) to converges uniformly on \(\mathcal{I}\).

Denote \(f_{h,h}\) by just \(f_h\). Using equicontinuity of \(f_h\) at a point \(r\) and pointwise convergence at \(r\) we can say

\begin{align*} |f_h(t) - f_h(r)| < \epsilon & \text{for } |t - r| < \delta(\epsilon)\\ |f_h(r) - f_{\ell}(r)| < \epsilon & \text{for } h, \ell \geq N(r,\epsilon)\\ |f_{\ell}(t) - f_{\ell}(r)| < \epsilon & \text{for } |t - r| < \delta(\epsilon) \end{align*}

The first and last conditions coming from equicontinuity and the second condition coming from pointwise convergence. Via triangle inequality we can bound \(|f_h(t) - f_{\ell}(t)|\). Then by equicontinuity, we can cover \(\mathcal{I}\) with \(\delta(\epsilon)\) balls and the above conditions will hold. By compactness, we can take a finite subcover where each ball is centered at some rational number. For each of these finite rational numbers, pick the largest \(N(r,\epsilon)\) and denote it \(N(\epsilon)\). Then for all \(h, \ell \geq N(\epsilon)\) we have that \(|f_{h}(t) - f_{\ell}(t)| < 3 \epsilon\) giving us uniform convergence on \(\mathcal{I}\).

2.2. Approximating with smooth functions

Next we approximate \(f\) with smooth functions (and thus Lipschitz). More precisely,

Lemma: Let \(\vec{f}(t,\vec{y})\) be continuous on a rectangle \(\mathcal{R}(a,b) = \mathcal{R}\). Then for every \(\epsilon > 0\) there exists \(\vec{F}_{\epsilon}(t,\vec{y})\) that is continuous on \(\mathcal{R}\), whose partial derivatives are all continuous on \(\mathcal{R}\) such that

  • \(|\vec{F}_{\epsilon}(t,\vec{y})| \leq \max_{(\tau, \vec{\eta}) \in \mathcal{R}} |\vec{f}(\tau, \vec{\eta})|\) for \((t,\vec{y}) \in \mathcal{R}\)
  • \(|\vec{F}_{\epsilon}(t,\vec{y}) - \vec{f}(t,\vec{y})| \leq \epsilon\) on \(\mathcal{R}\)

That is, we \(\epsilon\)-approximate \(\vec{f}\) with a smooth function \(\vec{F}_{\epsilon}\) on the rectangle \(\mathcal{R}\).

Proof.

Here, we simply give a proof idea. Let \(\vec{F}(t,\vec{y})\) extend \(\vec{f}\) to the entire spatial domain. That is, \(\vec{F}\) is defined for all \(\vec{y}\) and

\begin{align*} \vec{F}(t,\vec{y}) = \begin{cases} \vec{f}(t,\vec{y}) & \text{ on } \mathcal{R} \\ 0 & \text{ on } |t - t_0| \leq a, \ |\vec{y} - \vec{c}_0| \geq b+1 \end{cases} \end{align*}

This can be done using a partition of unity. Then let \(\Phi_{\delta}(\vec{y})\) be an approximation to the identity in \(\mathcal{L}^1(\mathbb{R}^n)\). Then define \(\vec{F}_{\epsilon}(t,\vec{y}) = \Phi_{\delta(\epsilon)}(\vec{y}) * \vec{F}(t, \vec{y})\).

2.3. Theorem

Theorem: If \(\vec{f}(t, \vec{y})\) in continous on a region \(\mathcal{R} = \{(t, \vec{y}) : |t - t_0| \leq a, \ |\vec{y} - \vec{c}_0| \leq b\}\) and \(M = \max_{\mathcal{R}}|\vec{f}(t,\vec{y})|\), there exists a function \(\vec{\phi}(t)\) such that

  • \(\vec{\phi}(t_0) = \vec{c}_0\)
  • \(\vec{\phi}'(t) = \vec{f}(t,\vec{\phi}(t))\) on \(|t - t_0 | \leq \alpha\)

where

\begin{align*} \alpha = \begin{cases} a & \text{ if } M = 0,\\ \min \{a, b/M\} & \text{ if } M > 0. \end{cases} \end{align*}

Proof.

Again, we mostly give a proof outline here. Let \(\vec{\phi}_{\epsilon}(t)\) be solutions to the smooth approximations \(\vec{F}_{\epsilon}\). The collection \(\mathcal{F} = \{\vec{\phi}_{\epsilon}(t) : \epsilon > 0\}\) is a bounded and equicontinuous set on \(|t - t_0| \leq \alpha\). This follows from the fact that \(|\vec{F}_{\epsilon}(t \vec{\phi}_{\epsilon})| \leq M\). Via Arzela-Ascoli we can get a subsequence \(\epsilon_{\ell} \to 0\) such that \(\vec{\phi}_{\epsilon_{\ell}}(t)\) converges uniformly on \(|t - t_0| \leq \alpha\) to some \(\vec{\phi}(t)\). It can be checked that this is indeed a solution to the ODE with \(\vec{f}\).
